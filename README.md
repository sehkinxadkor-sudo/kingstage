# Корали Сцены

Готовый шаблон сайта для набора в команду под перенос в Битрикс24.Сайты.

## Что внутри

- `index.html` - публичная страница: hero, анонсы, открытые роли, путь кандидата, анкета участника.
- `admin.html` - визуальная админка для заявок, задач постановки, основного описания, композиции таймлайна и анонсов.
- `styles.css`, `admin.css`, `script.js` - дизайн, адаптивность, микро-анимации и логика.
- `api/send-application.js` - пример серверного обработчика для отправки HTML-письма, создания лида в Битрикс24 и опциональной отправки в Telegram.
- `.env.example` - список переменных окружения без значений.

## Что редактируется в админке

- Основное описание сайта: заголовок, подзаголовок, блок о проекте, текст анкеты.
- Композиция блока «Что будет после отправки заявки»: лейбл, заголовок и 4 шага.
- Анонсы: добавление, редактирование и удаление.
- Заявки: просмотр, фильтр и смена статуса.
- Задачи постановки: создание и смена статуса.

## Перенос в Битрикс24

1. Создайте сайт в Битрикс24.Сайты.
2. Добавьте HTML-блок и перенесите разметку из `index.html`.
3. Подключите CSS из `styles.css` и JS из `script.js` через пользовательский HTML/JS-блок или настройки страницы.
4. Картинки из `assets` загрузите в медиатеку Битрикс24 и замените пути в контенте.
5. Для заявок используйте один из вариантов:
   - встроенную CRM-форму Битрикс24;
   - серверный обработчик `api/send-application.js`;
   - входящий вебхук Битрикс24, но только через сервер, не из фронтенда.

## Переменные окружения

Секреты не хранятся в коде. Создайте `.env` самостоятельно по примеру `.env.example`.

Ожидаемые переменные:

- `HR_EMAIL`
- `BITRIX24_WEBHOOK_URL`
- `BITRIX24_CRM_FORM_URL`
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_USER`
- `SMTP_PASSWORD`
- `SMTP_FROM`
